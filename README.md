# Udacity React Readable
This is the second project I've made for the Udacity nanodegree course.
Created by Henry Brown.
# Readable API Server
* Install and start the API server
    - `cd api-server`
    - `npm install`
    - `node server`

# Readable Frontend
* Install and start the frontend
    - `cd frontend`
		- `npm install frontend`
    - `npm start`
